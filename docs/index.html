eStundnzettl

Sichere Zeiterfassung. Lokal. Einfach.

eStundnzettl ermöglicht Dir eine präzise Arbeitszeiterfassung direkt auf Deinem Smartphone. Deine Daten werden nicht auf fremden Servern gespeichert, sondern bleiben in Deiner Hand.

  • Volle Kontrolle: Deine Daten bleiben lokal auf Deinem Gerät.
  • Einfacher Export: Generiere PDF-Berichte für Deine Unterlagen.
  • Google Drive Backup: Sichere Deine Daten optional in Deinem eigenen Cloud-Speicher.
